**SAT0RU: Enhanced Edition** is a high-performance evolution of the original cursed technique visualizer. While the core concept originated from [reinesana/SAT0RU](https://github.com/reinesana/SAT0RU), this version has been completely overhauled with advanced particle physics, expanded gesture recognition, and procedural audio to provide the ultimate Jujutsu Kaisen experience.

---

## 🚀 Key Enhancements in this Version

This version builds upon the original foundation with several major upgrades:
- **Expanded Technique Roster**: Added **6 new techniques** including *Fuga*, *World Cutting Slash*, *Black Flash*, *Boogie Woogie*, and *Nue*.
- **Refined Gesture Logic**: Implemented a high-precision `GestureEngine` with temporal smoothing to prevent flickering.
- **Advanced Particle Physics**: Increased particle count to **20,000+** with custom turbulence, vortex, and shockwave behaviors.
- **Procedural Sound Engine**: Added a real-time audio system that generates unique synthesized sounds for every technique.
- **Cinematic Overlays**: Implemented Kanji flash effects, screen shakes, and chromatic aberration for high-impact visuals.

## ✨ Core Features

- **Gesture Recognition Engine**: Powered by MediaPipe Hands, detecting 10+ unique hand signs with temporal smoothing.
- **Particle Physics**: Over 20,000 dynamic particles with custom behaviors (vortex, shockwaves, rifts, and lightning).
- **Cinematic Post-Processing**: Features UnrealBloom, Chromatic Aberration, and Grain filters for an anime-accurate aesthetic.
- **Procedural Sound Engine**: Real-time synthesized audio via Web Audio API that reacts to your movements.
- **Dynamic UI**: Kanji flash overlays and HUD elements that track "cursed energy" resonance.

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Edge, or Firefox).
- A functional webcam.
- Proper lighting for accurate hand tracking.

### Installation & Usage
1. **Clone the repository**:
   ```bash
   git clone https://github.com/reinesana/SAT0RU.git
   cd SAT0RU
   ```
2. **Run with Live Server**:
   If using VS Code, install the **Live Server** extension. Right-click `index.html` and select **"Open with Live Server"**.

3. **Allow Camera Access**:
   When prompted, grant the browser permission to use your webcam.
